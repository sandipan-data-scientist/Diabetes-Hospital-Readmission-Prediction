# Diabetes-Hospital-Readmission-Prediction
End-to-end ML pipeline on 10 years of US hospital diabetes data (130 hospitals, 71K patients from archives of US EHR in MIMIC-IV database). Covers data cleaning, EDA, weight imputation via Random Forest, SMOTE-balanced XGBoost readmission classifier, drug combination risk analysis, and a FastAPI inference endpoint containerised with Docker.
